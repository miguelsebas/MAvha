Se creo una interfaz grafica con los botones correspondientes a lo que pedian en el examen, la imagen su carga como multipart, en un array de bytes y se anexa a los demas requisitos, con permanencia en base de datos SQL anexo un archivo .bak que contiene la tabla creada para este proyecto, solo tienen que recuperar la base de datos atravez de el archivo que anexo. Esta Web se conecta a la API sugerida(MAvhaAPI). Se lo probo tambien con Fidle y Postman demostrando un correcto funcionamiento.

Se utilizo EntitieFramework, NewtonSoft, BootStrap4, HTML5, LinQ.

Para que desde la Web (MAvhaWeb) funcione ingresar a la Carpeta Servicios y modificar la variable baseUrl y poner el localhost que estan utilizando en la maquina que use para probar actualmente esta este: https://localhost:44323
